Introduction to CSS

CSS stands for Cascading Style Sheets

It's used to define the presentation of HTML elements

Selectors in CSS
Types Description Usage
Universal All the elements * { color: yellow; }
Element A specific element h1 { padding: 10px; }
Class A class of elements .p { margin: 20px; }
Id A unique id #first { font-size: 25px; }

Properties in CSS
Font Color Margin
Sets the size of the font Sets the color of the font Sets space outside the border
font-size: 20px; color: green; margin: 10px;
