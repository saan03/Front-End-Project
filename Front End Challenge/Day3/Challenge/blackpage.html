What is HTML?

  • HTML stands for Hyper Text Markup Language
  • It's the standard language used to create and design webpages
  • Backbone of web development

Basic HTML Tags

  1. Headings (h1-h6 tags) - HTML headings are titles or subtitles on a webpage
  2. Paragraphs (p tag) - Defines a paragraph of text
  3. Image (img tag) - Embeds an image in the web page

Tags and Attributes in HTML

HTML Tags

HTML tags are used to define elements on a webpage

HTML Attributes

Attributes provide additional information about HTML elements
